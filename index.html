I'm a Computer Science student at the University of Sheffield. I'm interested in application and network security, web development, and many other things like history, roleplaying games and board games.

Projects

With a team of five others, I helped make a Twitter management web app using Sinatra, SQLite, the Twitter API and Bootstrap. Although this was a relatively simple project, it was fun working with other developers, and a useful experience of agile development.

As part of an assignment, I made a static website (HTML, CSS, JavaScript) for a fictitious company. This was one of the first static websites I made start to finish.

At a hackathon I helped make a website that would use data on soldiers who died in the First World War to estimate which battle they died in. We planned to use the Wikipedia JSON API to get data about battles, and match that with data from another site. Although we didn't finish the site, I enjoyed myself learning about RESTful APIs and Python.
